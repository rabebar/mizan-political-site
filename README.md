# مؤسسة الميزان السياسي

موقع ثابت لمؤسسة الميزان السياسي للأبحاث والترجمة الإعلامية.

## النشر

يمكن نشر هذا المجلد مباشرة عبر GitHub Pages لأن ملف `index.html` موجود في جذر المستودع.

ويمكن نشره على Render كـ Static Site. يحتوي المستودع على ملف `render.yaml` جاهز:

- Runtime: `static`
- Build Command: `true`
- Publish Directory: `.`
