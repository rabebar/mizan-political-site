# مؤسسة الميزان السياسي

موقع مؤسسة الميزان السياسي للأبحاث والترجمة الإعلامية.

## النشر على Render

المشروع يعمل كخدمة Node لأن لوحة الإدارة تحتاج API لحفظ الأخبار والتحليلات.

إعدادات Render المطلوبة:

- Build Command: `npm install`
- Start Command: `npm start`
- Environment Variables:
  - `ADMIN_USERNAME`: اسم مستخدم السوبر أدمن
  - `ADMIN_PASSWORD`: كلمة سر السوبر أدمن
  - `SUPER_ADMIN_USERNAME`: اختياري، لترقية حساب أدمن موجود مسبقًا إلى سوبر أدمن

لا يوجد تسجيل أدمن مفتوح من الموقع. حساب السوبر أدمن يتم ضبطه من إعدادات Render فقط، ومن لوحة السوبر أدمن يمكن منح صلاحية أدمن تحرير أو سوبر أدمن لحسابات أخرى.

رابط لوحة الإدارة يكون عبر إضافة `#admin` إلى رابط الموقع.
