# مؤسسة الميزان السياسي

موقع مؤسسة الميزان السياسي للأبحاث والترجمة الإعلامية.

## النشر على Render

المشروع يعمل كخدمة Node لأن لوحة الإدارة تحتاج API لحفظ الأخبار والتحليلات.

إعدادات Render المطلوبة:

- Build Command: `npm install`
- Start Command: `npm start`
- Environment Variables:
  - `ADMIN_USERNAME`: اسم مستخدم الأدمن
  - `ADMIN_PASSWORD`: كلمة سر الأدمن

لا يوجد تسجيل أدمن مفتوح من الموقع. حساب الأدمن يتم ضبطه من إعدادات Render فقط.

رابط لوحة الإدارة يكون عبر إضافة `#admin` إلى رابط الموقع.
